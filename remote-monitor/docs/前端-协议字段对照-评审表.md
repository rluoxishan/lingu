> **用途**：监控页字段与 MQTT / 云平台 API 的对照与实现分档。  
> **关联协议**：`docs/protocol/吉狮-远程监控与控制协议-V1.0-draft.md`  
> **瑜权答复**：[确认清单-协议与改动.md](./确认清单-协议与改动.md)（含 §八 按钮速查表）

> **最后更新**：2026-07-01

## 实现分档说明

| 标记 | 含义 | 前端动作 |
| ---- | ---- | -------- |
| **V1.0** | 车端 **已有** 或云平台 **可立即对接** | 接真实 API / 遥测 |
| **协议·车端** | 瑜权清单 **已定义**，MCU/上位机 **待改代码** | payload 先写好，联调开关关闭 |
| **协议·中位机** | 瑜权清单 **已定义**，走 **中位机 UDP**，待导航/中位机实现 | 同上 |
| **待抓包** | 云平台 HTTP 开流，Swagger 不完整 | F12 抓 `/admin-api/camera/...` |
| **P2** | 可选或依赖中位机是否支持 | 继续 Mock |

**架构速览**（来自瑜权清单 §0）：

```
MCU 串口 A1/A2  →  速度/转向/急停/大灯/转向灯/超声/屏幕
中位机 UDP      →  视频推流/云台/作业/位姿/任务扩展
监控页开流      →  云平台 HTTP 一次请求 → playUrl（前端不涉及 2010004）
```

## 二、视频流（云平台 HTTP ↔ VideoPanel）

> 监控页：**一次 HTTP「开摄像头」** → 后台调厂商协议 → 响应 **各路 playUrl** → WebRTC。  
> **前端不涉及** MQTT 2010004（仅对外 MQTT 直连文档 §9）。

| 前端 CameraKey | view | 分档 | 远程驾驶 | 云台模式 | 备注 |
| -------------- | ---- | ---- | -------- | -------- | ---- |
| front | 1 FRONT | **V1.0** / **待抓包** | 主画面 | 辅窗 | view 1–4 车端已有 |
| right | 2 RIGHT | 同上 | 辅窗 | 辅窗 | |
| rear | 3 REAR | 同上 | 辅窗 | 辅窗 | |
| left | 4 LEFT | 同上 | 辅窗 | 辅窗 | |
| surround | 5 SURROUND | **协议·中位机** | 辅窗 | 不显示 | 是否列入 V1.0 待 Hasun+中位机 |
| ptz | 6 PTZ | **协议·中位机** | 不显示 | 主画面 | 或仅 GIMBAL 控制无独立 view |

## 三、远程控制（控制台 ↔ 2010005 / 2010002 / 2010008）

> 详细 payload 见 [确认清单 §八](./确认清单-协议与改动.md#八前端按钮协议速查表)。

### 3.1 底盘 WASD + 档位（2010005 · MCU A1）

| 前端操作 | payload | 分档 | 备注 |
| -------- | ------- | ---- | ---- |
| W/S/A/D | `command` + `speedLevel` + `seq` | **协议·车端** | 模式 A（监控页默认） |
| 模式 B 直连 | `speed`（m/s）+ `angle`（rad） | **协议·车端** | 车端已实现；车型兼容由车端配置 |
| 松开 | `command: "STOP"` | **协议·车端** | |
| P / N | `command: "STOP"` | **协议·车端** | MCU 无独立 N 档 |
| R | `BACKWARD` + `speedLevel: 1` | **协议·车端** | |
| D 档 | `FORWARD` + `speedLevel: 1` | **协议·车端** | |
| 下发频率 | 10–30 Hz，松开发 STOP | **协议·车端** | 不超过 30 Hz，不低于 10 Hz |
| 急停（遥控内） | 2010005 `EMERGENCY_STOP` | **协议·车端** | 建议 UI 急停走 2010008 |

**速度档参考**（模式 A，待联调微调）：FORWARD 1/2/3 → 0.3/0.6/1.0 m/s；BACKWARD → 0.15/0.3/0.5 m/s；LEFT/RIGHT → ±0.3 rad。

### 3.2 灯光 / 警示（2010002 flat · MCU A2）

| 前端按钮 | payload | 分档 | 备注 |
| -------- | ------- | ---- | ---- |
| 近光 | `headlights: 1` | **V1.0** | |
| 远光 | `headlights: 2` | **协议·车端** | MCU 已支持，代码需放开 2/3 |
| 关灯 | `headlights: 0` | **V1.0** | |
| 左转灯 | `turnSignals: 1` | **V1.0** | |
| 右转灯 | `turnSignals: 2` | **V1.0** | |
| 双闪/警示 | `turnSignals: 3` | **V1.0** | |
| 工作灯 | `headlights: 1` 代替 | **V1.0** | A2 无独立工作灯 bit |
| 刹车灯 | **2010008 急停** | **V1.0** | 不建议独立 LIGHT target（P2） |

### 3.3 作业（2010002 target=WORK · 中位机 UDP）

| 前端按钮 | payload | 分档 |
| -------- | ------- | ---- |
| 清扫 | `WORK SET { action: "SWEEP", on: true/false }` | **协议·中位机** |
| 洒水 | `action: "SPRAY"` | **协议·中位机** |
| 吸污 | `action: "SUCTION"` | **协议·中位机** |
| 卸料 | `action: "DUMP"` | **协议·中位机** |
| 全停 | `WORK STOP {}` | **协议·中位机** |
| 喇叭 | `AUDIO SET { action: "HORN", duration: 500 }` | **协议·中位机** |

### 3.4 急停（2010008 · MCU A1 · QoS 2 有回复）

| 前端按钮 | payload | 分档 |
| -------- | ------- | ---- |
| 急停 | `{ action: "EMERGENCY_STOP" }` | **协议·车端** |
| 解除 | `{ action: "RELEASE" }` | **协议·车端** |

### 3.5 云台（2010002 target=GIMBAL · 中位机 UDP）

| 前端操作 | payload | 分档 |
| -------- | ------- | ---- |
| 方向微调 | `GIMBAL MOVE_STEP { pan/tilt }` | **协议·中位机** |
| 变焦 +/- | `ZOOM_IN` / `ZOOM_OUT` | **协议·中位机** |
| 回中 | `HOME` | **协议·中位机** |
| 预置位 1–4 | `SET_PRESET` / `GOTO_PRESET` | **协议·中位机** |
| 停止 | `STOP` | **协议·中位机** |

> 云台 **不采用** 2010005 内嵌 cameraPan/Tilt（清单推荐独立 GIMBAL）。

## 五、评审结论（2026-07-01 更新）

### 瑜权 — 已答复（见 [确认清单](./确认清单-协议与改动.md)）

- [x] 云台：**2010002 GIMBAL**（§4.4），非 2010005 内嵌
- [x] MCU 边界：A2 仅 6 字节；作业/云台/喇叭 **不在 MCU**（830.pdf）
