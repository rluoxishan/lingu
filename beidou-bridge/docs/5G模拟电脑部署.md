# 5G 模拟电脑部署（先于中转机）

> **作用**：模拟车端 `LU2606000100`，向中转机 MQTT 发 **1010001**。  
> **不装**：Mosquitto、beidou-bridge 中转服务、`.env` 云账号。

---

## 你现在要做的事（5G 电脑）

### 1. 拷程序

1. U 盘插入 5G 电脑。  
2. 复制整个文件夹到本地（不要只拷 bat）：

```text
U 盘\beidou-bridge-site-20260702  →  D:\beidou-bridge
```

### 2. 装 Node.js

1. 双击 U 盘根目录 **`node-v24.18.0-x64.msi`**（或 node-v20）。  
2. 默认安装，**关闭并重新打开** cmd。  
3. 验证：

```bat
node -v
```

### 3. 环境检查

```bat
cd /d D:\beidou-bridge
CHECK-5g-sim.bat
```

全部 `[OK]` 即 **5G 电脑部署完成**。

---

## 中转机还没装时

- **不要**现在跑 `SIMULATE-vehicle.bat`（会连不上 1883，属正常）。  
- 5G 电脑这边到此为止即可。

---

## 中转机 Mosquitto + bridge 就绪后

1. 5G 网卡/模块插好，**ping 中转机 IP** 通。  
2. 运行：

```bat
cd /d D:\beidou-bridge
SIMULATE-vehicle.bat
```

输入中转机 IP。成功标志：

```text
[sim] MQTT 已连接
[sim] #1 pub workStatus=1 battery=... pos=...
```

或预设 IP 后运行：

```bat
set VEHICLE_MQTT_BROKER=mqtt://192.168.x.x:1883
SIMULATE-vehicle.bat
```

## 与中转机联调顺序（备忘）

| 顺序 | 机器 | 动作 |
|------|------|------|
| ① | **5G 电脑** | 本手册（已完成） |

| ③ | 5G 电脑 | `SIMULATE-vehicle.bat` |
| ④ | 中转机+北斗 | 接真北斗 register |
