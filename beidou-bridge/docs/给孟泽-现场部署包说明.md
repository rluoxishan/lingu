## 3. 到场安装（5 分钟）

在 **北斗提供的中转机** 上：

```powershell
# 1) 解压 zip 到例如 D:\beidou-bridge
# 2) 安装 Node.js 20+（若机器上没有）
# 3) 复制 .env 到解压目录
copy .env.example .env   # 若已有 .env 则跳过

# 4) 启动
cd D:\beidou-bridge
scripts\site-start.bat
```

另开窗口验证：

```powershell
Invoke-RestMethod http://127.0.0.1:8080/health
```

应返回 `status = up`。

## 5. 常见问题

| 现象 | 处理 |
|------|------|
| 没有 Node.js | 安装 Node 20 LTS 后重开 PowerShell |
| 登录云失败 | 检查 `.env` 账号 |
| position 全 0 | 商用版 + 定位；**不挡 B1** |
| 8080 被占用 | 联系我方或改 `config/site/server.yaml` |
