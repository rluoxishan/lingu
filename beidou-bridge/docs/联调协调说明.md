# 联调协调说明（对内 + 对外）

> **版本**：V1.0（2026-06-30）  
> **原则**：先向北斗确认 **联调时间 + 中转机环境**，再定我方到场部署；确认后打包给孟泽

---

## 1. 整体流程

```
① 向北斗确认联调时间 + 他们提供的中转机环境
        ↓
② 北斗回复：时间 / IP / 8080 / 回调 URL / 联系人 / 远程或现场方式
        ↓
③ 我方打包部署包 + .env（单独给孟泽）
        ↓
④ 联调当天：在中转机上安装 → Mock 预演 → 北斗 register → B1 验收
        ↓
⑤ 孟泽：LU2606000100 商用版在线
```

**中转机由北斗方提供**，我方带程序包到场（或远程）部署。
